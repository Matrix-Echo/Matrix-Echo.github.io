2、报错 TypeError: 'module' object is not callable

 

解决：

import module 和 from module import method有不同

若用前者，程序主体中要用 module.method 的格式调用方法；

若用后者，则直接用 method 调用方法

 

3、import的地方报错 unexpected indent (出现“意外缩进”)

 

解决：

这里不是指import语句本身有意外缩进，而是指要import的文件出现了意外缩进
